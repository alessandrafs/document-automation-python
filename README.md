# 📄 Document Automation with Python

This project automates the generation of documents using structured data from a CSV file.

---

## 🚀 Features
- Reads structured data from CSV file
- Automatically generates Word documents
- Reduces repetitive manual tasks
- Scalable for batch processing

---

## 🛠 Technologies
- Python
- pandas
- python-docx

---

## ▶️ How to run

1. Install dependencies:
```bash
pip install pandas python-docx
```

2. Run the script:
```bash
python main.py
```
